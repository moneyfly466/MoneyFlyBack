# 🐭💸 MoneyFly – Gestión de Gastos Hormiga  
**Proyecto educativo Full Stack con Spring Boot, React y Data Analysis en Python**

---

## 📌 Descripción del Proyecto  
**MoneyFly** es una aplicación educativa diseñada para ayudar a las personas a **identificar, analizar y controlar sus gastos hormiga** mediante una arquitectura moderna:

- **Frontend:** React ⚛️  
- **Backend:** Spring Boot ☕  
- **Data Analysis:** Python 🐍  
- **Base de datos:** (PostgreSQL / MySQL / MongoDB)

La aplicación centraliza los gastos diarios del usuario, genera estadísticas automáticas y visualizaciones basadas en patrones de consumo para mejorar la toma de decisiones financieras.

---

## 🎯 Objetivos del Proyecto  
- Registrar gastos hormiga de forma sencilla y rápida.  
- Analizar los datos usando Python (pandas, numpy, scikit-learn, etc.).  
- Mostrar insights financieros con gráficos dinámicos en el frontend.  
- Crear una API robusta y escalable con Spring Boot.  
- Integrar distintos ecosistemas tecnológicos en un único producto educativo.
